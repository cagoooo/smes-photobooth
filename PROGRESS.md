## ✅ 已完成（v1）

### 🎬 前端拍照 App
| 功能 | 狀態 |
|---|---|
| 即時相機預覽（自動鏡像、可切換鏡頭、記住選擇） | ✅ |
| 倒數自拍：立即 / 3 / 5 / 10 秒 + 倒數大數字 + 提示音 + 白閃快門 | ✅ |

| 結果彈窗：💾 存本機 / ☁️ 上傳雲端 / 📱 QR Code 下載 | ✅ |
| ⚙️ 設定：後端網址、相機鏡頭切換 | ✅ |
| 後端網址**內建預設**（任何裝置打開即可上傳，免設定） | ✅ |
| 頁尾署名（阿凱老師 → 石門教師頁） | ✅ |
| 🔄 Service Worker 版本更新通知（prompt-to-refresh） | ✅ |

| 端到端實測：上傳 + 公開連結 + QR 全通 | ✅ |
